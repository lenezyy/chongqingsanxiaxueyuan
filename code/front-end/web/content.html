我院成功开展xx实验项目
发布日期：2017-03-05 发布单位：教务处

农民农闲时可去哪里？去农家书屋。致富有门路吗？有，去农家书屋里找。农闲时候有地方去休闲吗？有，去农家书屋里消遣…… “泥腿子”成了农家书屋的主角 坐进敞亮的房屋，用满是老茧的手握笔认真记着笔记……这是笔者日前在天中街道铁渠庙村农家书屋见到的一幕场景。

走进天中街道办事处铁渠庙村的的农家书屋，一间约20平方米的小屋，书架上整齐地摆放着一排排图书，书桌椅凳一应俱全，此外，还配备了电脑、音响等信息共享工程设备。据图书管理员帅小容说，该书屋藏书1500册，其中多数是农村很实用的农用知识类型的书，如经济作物病虫草害防治新技术、果树林木嫁接技术手册、科学养猪指南、优质鸡肉鸭饲养技术。还有部分中老年村民比较关心的保健、食疗类型的书，以及小朋友们喜欢的益智类、漫画类书籍。 书屋里，不时有农民朋友聚集在农家书屋里，或挑上一本自己满意的书细细琢磨，或观看科技光盘，一同探讨技术上的难题。日前，笔者看到一位中年农民一边翻书《优质鸡肉鸭饲养技术》，一边在厚厚的本子上记笔记，他是该村的一位养殖大户，“现在咱泥腿子也成文化人了！”村民说，他说：“平时很忙碌，难得有时间看书，如今利用正月这个空闲，多学点科学知识，让今年的收成更好些。” 农家书屋引导村民农闲新风尚

农家书屋之所以受欢迎，不仅是因为它让老百姓有了学习和掌握致富技能的机会，更是让广大老百姓通过农家书屋这个载体，在农闲之余开展多种形式的培训和文艺活动，大大丰富了农村的业余文化生活。据铁渠庙村党支部书记曹明臣介绍，为了提高老百姓的阅读积极性，村里会经常在农家书屋举办读书活动；技术人员下乡培训时，也充分利用了农家书屋的图书，因材施教，对老百姓进行教授和辅导。让村干部意想不到的是，自从开设了农家书屋，这里逐渐成为了村里搞文体活动的场地，村民笔下地时就来到这里休闲，“我们村有不少"能人"，吹拉弹唱、打球练操都能来几手！” “金屋银屋，比不上农家书屋；金山银山，比不上文化书山。”17届六中全会以后，天中街道将农家书屋的建设作为丰富群众文化生活的惠民工程，至今，农家书屋在村（社）的覆盖率已达到95%，按照建管并重、合理规划、就近就便的原则，兴建了十余处书屋，有效解决了群众看书难的问题，培养了基层群众的文化自觉和自信，使农家书屋成为基层群众学科学、用科学的“取经屋”、“加油站”。“村里学习科技知识的风气时兴起来，老百姓打牌消遣、打架斗殴的现象少了。”曹明臣自豪地说，农家书屋的人气兴旺，相信科学、崇尚文明的新风尚正在逐步形成。
